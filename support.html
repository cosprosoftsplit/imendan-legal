Frequently Asked Questions

How do I search for a name day?

Use the search bar on the Library tab to find any name. You can search in Croatian or English.

How do I set up reminders?

Go to Settings and enable notifications. You can choose the time of day you'd like to be reminded about upcoming name days.

The app doesn't require an internet connection?

Correct! Imendan works completely offline. All name day data is included in the app.

I found an incorrect name day. How can I report it?

Please email us at the address below and we'll correct it in the next update.
